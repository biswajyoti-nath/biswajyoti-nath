# Hi, I'm Biswajyoti Nath 👋

I'm an undergraduate Computer Science researcher exploring the intersection of **Machine Learning, Symbolic Regression, Evolutionary Computation, and Interpretable Machine Learning**.

My research explores how **representation, constraints, and search strategies** influence the behavior of symbolic learning systems. I enjoy studying algorithms from first principles, designing reproducible computational experiments, and building research software that bridges theory with practical implementation.

## 🚀 Featured Projects

### 🧪 **[EML Framework](https://github.com/biswajyoti-nath/eml-framework)**

A reproducible Python framework for studying grammar-constrained symbolic regression through validity-aware structural tracking and multi-trial empirical evaluation.

### 🚌 **[Pather Saathi](https://github.com/Subinoy-Nath/pathersaathi)**
A production-oriented fleet booking platform built with **Next.js 15, PostgreSQL, Supabase, and Upstash Redis**, emphasizing secure authentication, Row-Level Security (RLS), backend architecture, and scalable booking workflows.

### 🤖 **[Startup LLM Content Pipeline](https://github.com/biswajyoti-nath/startup-llm-content-pipeline)**

A multi-agent pipeline utilizing Claude, Gemini, and Tavily to transform unstructured conversational data into structured narratives and publishable content.
